Course Objective
The purpose of this course is to give students a practical introduction to databases and Web app development. Topics we will study in this course include:
Database
  • Terminology and applications.
  • Creating, query and updating databases
  • The entity-relationship model for database design.
Web documents and applications
  • Static and interactive documents
  • Web servers and dynamically server generated content
  • Web application development and interface with databases
Tutorials
There are weekly tutorials. You are expected to attend and participate.
Calendar

Assignments
